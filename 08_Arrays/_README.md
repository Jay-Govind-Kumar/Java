1. Array:
   A collection of elements of the same type stored in contiguous memory locations.

2. Array Declaration:
      ```
         int arr[] = new int[5];
      ```

3. Array Declaration & Initialization:
   ```
      int arr[] = {1, 2, 3, 4, 5};
      int arr[] = new int[] {1, 2, 3, 4, 5};
   ```

4. Accessing Array Elements:
   ```
      arr[i];
   ```

5. Updating Array Elements:
   ```
      arr[i] = value;
   ```
