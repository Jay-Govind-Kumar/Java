- Learned about `Hashing` in Java.

## Introduction

- Hashing is a technique used to store and retrieve data from a data structure.
- It is a way of mapping keys to values.

## Types of Hashing

- `Map`
  - HashMap
  - LinkedHashMap
  - TreeMap
- `Set`
  - HashSet
  - LinkedHashSet
  - TreeSet
